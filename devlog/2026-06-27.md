# 开发日志 - 2026年06月27日

## 完成事项

### 1. Docker 容器化部署 ✅
- `Dockerfile.backend`: Python 3.10 + Poetry 多阶段构建
- `Dockerfile.frontend`: Node 18 构建 + Nginx 运行
- `docker-compose.yml`: MySQL + Redis + Backend + Frontend 一体化编排
- `nginx/nginx.docker.conf`: Docker 环境 Nginx 反向代理（SSE 支持）
- `.env.docker`: Docker 环境变量模板
- `docker-entrypoint.sh`: MySQL 等待 → 迁移 → 启动
- `.dockerignore`: 构建排除规则

### 2. 安全修复 ✅
- 移除 `config.py` 中硬编码的 `DEEPSEEK_API_KEY`
- `DATABASE_URL` 支持环境变量覆盖（测试/生产灵活切换）
- `DB_POOL_SIZE` / `DB_MAX_OVERFLOW` / `DB_ECHO` 可配置

### 3. 测试体系建设 ✅
- pytest + pytest-asyncio + pytest-cov 框架
- SQLite 内存数据库作为测试后端
- **26 个测试用例**全部通过：
  - 健康检查 (1)
  - 密码哈希/验证 (3)
  - JWT 创建/验证 (4)
  - 注册/登录 API (6)
  - 部门 CRUD (6)
  - 文档管理 (4)
  - 权限验证 (2)

### 4. CI/CD Pipeline ✅
- `.github/workflows/ci.yml`: Backend (lint→test) / Frontend (build)
- `.github/workflows/docker-build.yml`: Docker 镜像构建缓存

### 5. 代码质量工具 ✅
- Ruff 配置（py310 target, isort, pyflakes, bugbear, pyupgrade）
- `.pre-commit-config.yaml`: trailing-whitespace + ruff + prettier
- `frontend/.prettierrc`: Prettier 格式化配置

### 6. 文档完善 ✅
- `README.md`: 架构图(Mermaid) + API一览 + Docker快速开始
- `LICENSE`: MIT License
- `CHANGELOG.md`: v0.1.0 版本记录
- `Makefile`: dev/test/lint/docker 等便捷命令

### 7. 安全加固 ✅
- 轻量级 Rate Limiter（登录 10次/分钟，注册 5次/分钟）
- CORS 严格配置（仅允许 Authorization/Content-Type headers）
- CORS_ORIGINS 环境变量可配置

### 8. 数据库配置优化 ✅
- `config.py`: `ASYNC_DATABASE_URL` 属性替代原 `DATABASE_URL`
- `DATABASE_URL` 改为可环境变量覆盖的字段
- `database.py`: 连接池参数可配置
