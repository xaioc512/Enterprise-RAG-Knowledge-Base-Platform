### 阶段三：文档管理 ✅
- **RAG Pipeline**：上传→解析→分块→向量化→Chroma存储
- **Loader**：PDF(PyPDF2) / Word(python-docx) / Markdown / TXT
- **Splitter**：RecursiveCharacterTextSplitter (chunk=500, overlap=50)
- **Embedder**：Chroma ONNX all-MiniLM-L6-v2（本地，无需API）
- **Retriever**：Chroma 原生 API（add/delete/search）

### 阶段四：AI问答核心 ✅
- **Generator**：DeepSeek Chat API 流式生成（AsyncOpenAI SDK）
- **Pipeline**：检索→构建Prompt→流式生成→SSE输出
- **SSE流式**：`POST /api/chat/send` → text/event-stream
- **对话管理**：CRUD /api/conversations/
- **反馈**：POST /api/feedback/（点赞/点踩）
- **标题生成**：首条消息用DeepSeek生成对话标题
- **两阶段提交**：流中断不影响用户消息保存

## 注意事项
- 首次部署需先创建MySQL数据库 + 配置.env + 运行alembic迁移
- Chroma首次启动会下载ONNX模型（~80MB），启动稍慢
- 开发模式：`npm run dev`（前端）+ `poetry run uvicorn`（后端）
- 生产模式：`npm run build` + Nginx + Uvicorn
