# 06 - 数据库设计规范

> 版本：v1.0 | 日期：2026-06-08

---

## 1. MySQL 表结构

### 1.1 用户表 `users`

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INT | PK, AUTO_INCREMENT | 主键 |
| username | VARCHAR(50) | UNIQUE, NOT NULL | 用户名 |
| password_hash | VARCHAR(255) | NOT NULL | bcrypt哈希 |
| email | VARCHAR(100) | UNIQUE | 邮箱 |
| role | ENUM('admin','user') | NOT NULL, DEFAULT 'user' | 角色 |
| is_active | BOOLEAN | DEFAULT TRUE | 启用状态 |
| created_at | DATETIME | DEFAULT NOW() | 创建时间 |
| updated_at | DATETIME | ON UPDATE NOW() | 更新时间 |

### 1.2 知识分类表 `categories`

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INT | PK, AUTO_INCREMENT | 主键 |
| name | VARCHAR(100) | UNIQUE, NOT NULL | 分类名称 |
| description | TEXT | | 分类描述 |
| sort_order | INT | DEFAULT 0 | 排序 |
| created_at | DATETIME | DEFAULT NOW() | |

### 1.3 文档表 `documents`

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INT | PK, AUTO_INCREMENT | 主键 |
| title | VARCHAR(255) | NOT NULL | 文档标题 |
| category_id | INT | FK → categories.id | 分类 |
| file_type | VARCHAR(20) | NOT NULL | pdf/docx/md/txt |
| file_path | VARCHAR(500) | NOT NULL | 存储路径 |
| file_size | INT | | 字节数 |
| chunk_count | INT | DEFAULT 0 | 分块数量 |
| status | ENUM('processing','done','error') | DEFAULT 'processing' | 处理状态 |
| error_message | TEXT | | 错误信息 |
| uploaded_by | INT | FK → users.id | 上传者 |
| created_at | DATETIME | DEFAULT NOW() | |
| updated_at | DATETIME | ON UPDATE NOW() | |

### 1.4 文档分块表 `document_chunks`

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INT | PK, AUTO_INCREMENT | 主键 |
| document_id | INT | FK → documents.id, CASCADE | 关联文档 |
| chunk_index | INT | NOT NULL | 分块序号 |
| content | TEXT | NOT NULL | 文本内容 |
| chunk_hash | VARCHAR(64) | | 内容哈希（去重） |
| created_at | DATETIME | DEFAULT NOW() | |

### 1.5 对话表 `conversations`

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | CHAR(36) | PK, UUID | 主键 |
| user_id | INT | FK → users.id | 所属用户 |
| title | VARCHAR(255) | DEFAULT '新对话' | 对话标题 |
| created_at | DATETIME | DEFAULT NOW() | |
| updated_at | DATETIME | ON UPDATE NOW() | |

### 1.6 消息表 `messages`

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | CHAR(36) | PK, UUID | 主键 |
| conversation_id | CHAR(36) | FK → conversations.id, CASCADE | 关联对话 |
| role | ENUM('user','assistant') | NOT NULL | 角色 |
| content | TEXT | NOT NULL | 消息内容 |
| sources | JSON | | 引用来源 [{title, chunk_id, content_preview}] |
| created_at | DATETIME | DEFAULT NOW() | |

### 1.7 反馈表 `feedbacks`

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INT | PK, AUTO_INCREMENT | 主键 |
| message_id | CHAR(36) | FK → messages.id | 关联消息 |
| user_id | INT | FK → users.id | 反馈人 |
| rating | ENUM('like','dislike') | NOT NULL | 评价 |
| created_at | DATETIME | DEFAULT NOW() | |

## 2. Redis 数据设计

| Key | 类型 | 用途 | TTL |
|-----|------|------|-----|
| `jwt:blacklist:{token_hash}` | String | JWT登出黑名单 | Token过期时间 |
| `user:{user_id}:session` | Hash | 用户会话缓存 | 24h |
| `celery:task:*` | — | Celery Broker/Backend | 自动管理 |
