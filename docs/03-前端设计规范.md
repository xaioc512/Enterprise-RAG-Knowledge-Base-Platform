# 03 - 前端设计规范

> 版本：v1.0 | 日期：2026-06-08 | **强制执行**

---

## 核心原则（来自 skill.md）

本项目前端开发**必须严格遵守** `skill.md`（frontend-design skill）中的所有规范。

### 设计思维

编码前必须明确：
- **Purpose**：企业私有知识库AI问答平台 — 内部员工效率工具
- **Tone**：专业、高效、可靠的企业级工具感 — 选择**精致极简主义**（Refined Minimalism）风格
- **Constraints**：Vue 3 + Element Plus，需保证功能性和可访问性
- **Differentiation**：区别于通用ChatGPT式界面，打造企业专属的专业工具质感

### 铁律

1. **禁止使用通用AI美学**：Inter/Roboto/Arial等字体，紫色渐变+白色背景等套路配色
2. **字体选择**：使用独特、有品位的字体组合（展示字体 + 正文字体配对）
3. **色彩**：主导色+锐利强调色，避免均匀分布的调色板
4. **动效**：CSS优先，聚焦关键瞬间（页面加载的交错入场动画），滚动触发和hover惊喜
5. **空间**：不对称布局、叠层、对角流动、破格元素
6. **背景**：氛围和深度，渐变网格/噪点纹理/几何图案/透明叠层
7. **细节**：装饰边框、自定义光标、颗粒叠加效果

### 每次实现都应当不同

不得在不同页面/组件中重复相同的设计模式。每个生成都应有独特的审美决策。

---

## 具体执行规范

### 1. 设计方向：精致极简（Refined Minimalism）

- 大面积留白，精准间距控制
- 克制的色彩使用：深色主基调 + 单一强调色的变化
- 精致的微交互和动效
- 排版驱动设计

### 2. 色彩方案

```css
:root {
  /* 主色调 — 暗色专业风 */
  --bg-primary: #0d1117;
  --bg-secondary: #161b22;
  --bg-tertiary: #21262d;
  --text-primary: #e6edf3;
  --text-secondary: #8b949e;
  --accent: #58a6ff;        /* 蓝色强调 */
  --accent-hover: #79c0ff;
  --border: #30363d;
  --success: #3fb950;
  --warning: #d29922;
  --danger: #f85149;
}
```

### 3. 字体系统

- **展示字体**：DM Serif Display（标题、品牌标识）
- **正文字体**：Inter（非通用替代？选用更独特的，如 Source Sans 3 或 Public Sans）
- **代码/等宽**：JetBrains Mono

### 4. 间距系统

基于 4px 基准网格：4, 8, 12, 16, 20, 24, 32, 48, 64, 96

### 5. 动效规范

- 页面加载：staggered reveal（animation-delay级联）
- 消息出现：从下淡入，translateY(8px) → 0
- Hover：微妙缩放(1.02)或边框发光
- 流式文字：打字机光标闪烁效果

### 6. Element Plus 覆写

在 Element Plus 基础上做深度主题定制，不能使用默认样式：
- 统一的暗色主题
- 圆角、阴影、边框颜色全部自定义
- 消息气泡、卡片、输入框等关键组件自行设计
