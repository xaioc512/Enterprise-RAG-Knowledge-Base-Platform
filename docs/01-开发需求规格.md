# 01 - 开发需求规格

> 版本：v1.0 | 日期：2026-06-08 | 状态：已确认

---

## 1. 项目目标

构建企业内部私有化部署的生成式AI知识问答平台，基于RAG技术实现制度规范、工艺流程、流程文档、技术文档的AI智能问答。

## 2. 用户角色

| 角色 | 权限 |
|------|------|
| 管理员 (admin) | 文档上传/管理、知识分类管理、用户管理、系统设置 |
| 普通用户 (user) | AI问答、对话历史查看、反馈（点赞/点踩）、文档预览 |

所有用户可访问全部知识库（无部门隔离）。

## 3. 功能清单

### 3.1 管理后台
- **知识分类管理**：CRUD，四大预设分类（制度规范/工艺流程/流程文档/技术文档）
- **文档管理**：
  - 手动上传 PDF/Word(.docx)/Markdown/TXT
  - 自动解析 → 分块 → 向量嵌入 → Chroma入库
  - 文档预览（PDF内嵌/Markdown渲染/文本展示）
  - 文档删除、重新处理

### 3.2 用户端
- **AI智能问答**：多轮对话、RAG检索增强、流式响应
- **引用溯源**：AI回答标注来源文档和片段
- **对话历史**：按时间倒序、可进入继续对话
- **反馈机制**：点赞/点踩

### 3.3 通用
- 用户注册/登录/信息修改
- 用户管理（管理员操作）

## 4. 非功能需求

- 响应时间：问答响应 < 10s（含RAG检索+LLM生成）
- 文档处理：异步后台处理，不阻塞上传操作
- 安全性：JWT鉴权，密码哈希存储，SQL注入防护
- 日志：结构化JSON日志，便于排查

## 5. MVP边界

**包含**：上述全部功能模块
**不含**：全文搜索、数据看板、增量同步、模型切换、审计日志（后续迭代）
