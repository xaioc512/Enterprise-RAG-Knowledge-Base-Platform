# 08 - 开发执行步骤

> 版本：v1.0 | 日期：2026-06-08 | 实施中

---

## 执行原则

1. **小步迭代**：每个步骤完成后验证，确认无误再进入下一步
2. **零容忍Bug**：编译/运行时错误立即修复，不累积
3. **每日日志**：每天更新 `devlog/` 中的开发日志
4. **严格规范**：前端遵守 `skill.md` + `docs/03-前端设计规范.md`，后端遵守 `docs/04-后端开发规范.md`
